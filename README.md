# my-homepage
ホームページ

## 環境変数
`.env.sample`をコピーして`.env`を同じディレクトリに作成します。

- `TZ`
  - タイムゾーン
- `NODE_ENV`
  - 実行環境
    - 本番環境→`production`
    - 開発環境→`development`
- `CONTENTFUL_SPACE_ID`
  - ContentfulのスペースID
- `CONTENTFUL_TOKEN`
  - ContentfulのAPIトークン
    - 本番環境→Content Delivery APIのトークン
    - 開発環境→Content Preview APIのトークン
